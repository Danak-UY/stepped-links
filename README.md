# Stepped Links

Stepped Links is a Raycast extension that allows you to define and use multi-step links. This is useful for navigating through websites or applications that require multiple clicks or actions to reach a specific destination. You can define a series of steps, and the extension will execute them sequentially, opening the final link in your default browser.

## Commands

The extension provides the following commands:

- **Create New Step**: Allows you to create a new stepped link.
- **Import Steps Config**: Import a JSON file with your stepped links configuration.
- **Export Steps Config**: Export your current stepped links configuration to a JSON file.
- **Import from Browser Bookmarks**: Import an HTML file with browser bookmarks as stepped links.
- **Export as Browser Bookmarks**: Export your stepped links as an HTML bookmarks file.
- **Import from Quick Links**: Import an HTML file with Quick Links as stepped links.
- **Export as Quick Links**: Export your stepped links as a Quick Links JSON file.
- **Search stepped link**: Search and open a specific stepped link.

## Preferences

The extension offers the following preferences to customize its behavior:

- **Separators**: Defines the characters used to separate steps in a query (default: `-`).
- **Query wildcard**: Specifies the wildcard used to replace the query in a URL (default: `{{q}}`).
- **Default browser**: Sets the default browser to open the final link.
- **Default config app**: Sets the default application to open and edit the configuration file.
